# Project Overview

This project is a comprehensive application designed for parents to manage their children's activities and access various resources. It includes features such as tracking children's locations, managing schedules, and accessing educational content.

## Setup Instructions

### Prerequisites
- Node.js (v14 or above)
- Android Studio
- Firebase account

### Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/singhd7426/Parent-app.git
   ```
2. Install the required dependencies:
   ```bash
   cd Parent-app/web
   npm install
   ```
3. Set up the Android environment:
   ```bash
   cd Parent-app/android
   // Open and configure with Android Studio
   ```
4. Run the application:
   ```bash
   npm start
   ```
